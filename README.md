# Personal Resume - Aram Harb

Welcome to my personal resume website — a modern, responsive project designed to present my background, technical skills, and development journey as a **Junior Web Developer** in a clean and engaging way.

This project reflects not only my experience and education, but also my attention to design, usability, and professional presentation.

## Preview

Open `index.html` in any modern browser to explore the resume locally.

## About This Project

This resume project was created as a web-based version of my professional profile, combining clear structure, responsive design, and interactive elements.

In addition to this portfolio-style resume, I also provided:

- **My original CV**
- **An ATS-friendly resume template**

Using both, an **ATS-optimized resume version** was prepared to better align with Applicant Tracking System (ATS) standards commonly used by recruiters and hiring platforms.

This means the project supports two important goals:

- presenting my profile in a visually appealing web format
- maintaining a professional resume version that performs well in ATS screening processes

## Features

- **Responsive Design** — A clean two-column layout that adapts smoothly to different screen sizes
- **Interactive Skill Bars** — Animated indicators to highlight technical proficiency
- **Print Ready** — Built-in print functionality for exporting to PDF or paper
- **Portfolio Links** — Direct access to selected GitHub repositories and work samples
- **Professional Resume Support** — Includes the foundation for an ATS-friendly resume version based on a provided template and CV

## Tech Stack

- **HTML5**
- **CSS3** — Custom styling, layout, and visual effects
- **JavaScript** — Interactivity and print functionality
- **Font Awesome** — Icons and visual enhancement
