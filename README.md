# Personal Resume - Aram Harb

A modern, responsive web-based resume showcasing my skills and experience as a Junior Web Developer.

## Preview

Open `index.html` in any modern browser to view the resume.

## Features

- **Responsive Design** - Clean two-column layout that adapts to different screen sizes
- **Interactive Skill Bars** - Animated progress bars displaying technical proficiency
- **Print Ready** - Built-in print button for PDF/paper exports
- **Portfolio Links** - Direct links to GitHub project repositories

## Tech Stack

- HTML5
- CSS3 (Custom styling with progress animations)
- JavaScript (Interactivity and print functionality)
- Font Awesome (Icons)
